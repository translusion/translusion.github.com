    An introduction to Swift for Python developers

    Sat, Nov 8, 2014
    While Swift fundamentally isn’t very much like Python, it looks as if it has a number of similarities on the surface. For instance both have different REPL environments. These are great ways of learning how to code. I can use the Python REPL as a calculator: >>> 2 + 3 5 >>> 10 + 2 - 5 7 >>> 3.5 * 2 7.0 >>> We can do much the same with Swift: 1> 2 + 3 $R0: Int = 5 2> 10 + 2 - 5 $R1: Int = 7 3> 3.5 * 2 $R2: Double = 7 4> Although this is quite similar, we can already get a hint here that Swift is different and cares a lot more about type.
