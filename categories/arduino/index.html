    Getting started with AVR programming

    Fri, Mar 1, 2013
    After Arduino So you tried Arduino programming. It is a great way to get introduced to Microcontrollers. But if you are actually an experienced programmer it doesn’t quite feel like the real thing. It is sort of like learning BASIC when you know there is this other thing real programmers use which is C. Arduino is built around the AVR microcontroller. It simplifies a lot by providing a USB port on the board itself and having a loaded program to load your program into the memory of the AVR microcontroller.
